# 🏛️ Ellis Island Screenplay

**Historically accurate screenplay about Italian immigrants arriving at Ellis Island, circa January 1905.**

## Project Overview

This repository organizes all research notes, historical references, and screenplay drafts for a screenplay set in January 1905, following a southern Italian immigrant family through the Ellis Island processing experience and their first days in New York's Little Italy.

## Key Research Findings

### Astronomy (January 15, New York — proxy for 1905)
- Sunrise: **7:18 AM** | Sunset: **4:52 PM** (~9½ hours daylight)
- Moon: **Waxing Crescent, 18% illumination** — rises 10:00 AM, sets 9:38 PM
- *Perfect symbolism: a sliver of light emerging from darkness*

### Weather (NYC, January circa 1905)
- Highs: ~35–38°F | Lows: ~22–26°F | Frequent cold snaps to 10°F
- Harbor fog common at dawn; ice floes on rivers
- Coal-smoke haze over Manhattan (no clean-air regulations)
- Snow cover likely persistent; 8–12" average January snowfall

### Italian Immigrant Context (1905)
- Peak decade: 2+ million Italians arrived (1900–1910)
- Predominantly from southern Italy (Sicily, Calabria, Campania)
- Steerage voyage: 10–14 days, cramped and unsanitary
- Ellis Island: "L'Isola delle Lacrime" (Island of Tears)
- Most feared: the trachoma eye exam with buttonhook
- Little Italy centered on Mulberry Street; tenement housing; padrone labor system

## How to Use This Repo

1. **Read the research files** in `research/` before writing any scene
2. **Use the outline template** in `drafts/` as your structural backbone
3. **Add new research** as separate files in `research/` (e.g., `ship-design.md`, `tenement-layouts.md`)
4. **Write draft scenes** in `drafts/` with clear act/scene numbering
5. **Cross-reference** — tag each scene with the research file it draws from
