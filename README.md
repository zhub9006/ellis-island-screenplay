# Ellis Island Screenplay

A historically accurate screenplay about Italian immigrants arriving at Ellis Island, New York, circa 1905.

## Project Overview

This project documents the research and creative process for a screenplay set in early 1905, following an Italian family's journey from Southern Italy through Ellis Island and into the streets of New York's Little Italy neighborhood.

## Key Historical Context

- **Time Period:** January 1905
- **Location:** Ellis Island → Lower Manhattan (Little Italy, Mulberry Street area)
- **Subject:** Southern Italian immigration (primarily from Campania, Calabria, Sicily)
- **Peak Immigration:** 1900–1914 was the peak era; ~1,900 people processed daily at Ellis Island
